# EcoRoute Frontend

Next.js React frontend for EcoRoute sustainable logistics platform.

## Quick Start

```bash
npm install
npm run dev
```

Frontend runs on `http://localhost:3000`

## Pages

- **Home** (`/`) - Landing page
- **Route Planner** (`/planner`) - Interactive route optimization
- **Dashboard** (`/dashboard`) - Analytics and metrics
- **Fleet Management** (`/fleet`) - Vehicle management
- **Pricing** (`/pricing`) - Subscription plans
- **Contact** (`/contact`) - Contact form

## Environment Variables

```
NEXT_PUBLIC_MAPBOX_TOKEN=your_mapbox_token
NEXT_PUBLIC_API_URL=http://localhost:5000
```

## Building & Deployment

```bash
npm run build   # Build for production
npm start       # Start production server
```

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- Mapbox GL
- Recharts
- Lucide React
